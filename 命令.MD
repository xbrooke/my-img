## py脚本需要的环境

以下是你需要安装的库：

1. **`pyperclip`**：用于复制到剪贴板的 Markdown 链接。

安装命令：
```bash
pip install pyperclip
```

2. **`ffmpeg`**：用于转换图片格式，必须手动安装。

安装步骤：
- 下载并安装 FFmpeg：[FFmpeg 官网](https://ffmpeg.org/download.html)
- 将 FFmpeg 添加到系统环境变量中。

其他库（`logging`, `pathlib`, `subprocess`）是 Python 内置的，无需额外安装。

这就是你需要安装的所有内容。
